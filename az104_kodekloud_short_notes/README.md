## 2026 alignment reminders

The course covers the core domains well. For the current exam, also make sure you can:

- Create and manage an Azure Container Registry (ACR).
- Configure storage object replication, stored access policies, blob/file soft delete, snapshots, and blob versioning.
- Configure VM encryption at host and move VMs between supported scopes or regions.
- Use Azure Monitor Insights, alert processing rules, Network Watcher, and Connection Monitor.
- Distinguish a Recovery Services vault from an Azure Backup vault.
- Use Azure Monitor Agent (AMA) with Data Collection Rules (DCRs). The legacy Log Analytics/MMA agent is retired.

## Suggested study loop

1. Read the module note before watching the module.
2. Complete the related lab without copying steps.
3. Re-read only the **Exam cues** section.
4. Write down mistakes from the quiz or mock exam.
5. Rebuild one small environment from memory: identity + RBAC, VNet + NSG, storage, VM, backup, and monitoring.
